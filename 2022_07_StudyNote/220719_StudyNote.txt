1. JavaScript 강의 시청
	- document.body 이런 식으로 document 하위 객체 중 태그 이름으로 직접 가져올 수 있는 것은 body, head, title과 같은
	중요 태그들만 가져올 수 있다.
	(ex : document.div 와 같은 형식은 X, 이 때는 querySelector와 같은 함수로 가져올 수 있다.)

	- toggle 함수
	매개변수로 넘겨준 값이 현재 있는지 판단하여 있으면 제거, 없으면 추가해 주는 함수로 js로 html 요소의 클래스 이름 추가 등의
	상황에 유용하게 사용할 수 있다.

	- html에서 <script>와 <script type="text/javascript"> 의 차이점
	예전에는 text로 js가 쓰일 것이라는 것을 알려 주는 역할이었지만 html 최근 버전부터는 <script> 에도 기본 내장으로 text/javascript가
	있기 때문에 굳이 적어주지 않아도 차이점은 없지만, 페이지 간의 하위 호환성을 위해 적어 주는 것이 좋다.

	- onsubmit 이벤트
	form 형식에서 input에 값 입력 후 엔터를 누르거나 버튼을 눌렀을 때 나오는 이벤트로, 기본 형태는 입력한 값을 지정한 위치(서버)에
	보내면서 페이지를 새로고침한다.
		- event 함수의 첫 번째 인자에서 preventDefault 함수
		발생할 이벤트에서 기본 동작이 실행되지 않게 막아 준다.
		ex : submit 이벤트 발생 시 호출되는 함수 내에서 preventDefault 함수를 실행시키면 submit 이벤트의 기본 동작인
		페이지 새로 고침이 일어나지 않게 된다.

	- localStorage
	브라우저에 데이터를 저장할 수 있도록 하는 Web API이다.
	(브라우저를 종료해도 데이터가 없어지지 않으며, 브라우저를 종료했을 때 데이터가 삭제되는 것은 sessionStorage라고 따로 있다.)

	- 반복되는 특정 값(string, number 값 등....)이 있다면 변수로 지정해 놓고 변수명으로 사용하는 것이 오류 발생률, 유지 보수 등의 측면에서
	봤을 때 더 효율적이다.
	=> 이것과 마찬가지의 개념으로, 반복되는 작업이 있다면 해당 작업을 함수로 묶어 사용하는 것이 효율적이다.
